# Microsoft Store

mdviewx is distributed to the Store as an **MSIX** built from the WinAppSDK head
(`net10.0-windows10.0.26100`). The `release` workflow produces an **unsigned** MSIX
(`-p:AppxPackageSigningEnabled=false`) — the Store re-signs on submission.

## One-time setup (Partner Center)

1. Create a [Partner Center](https://partner.microsoft.com/dashboard) developer account.
2. **Reserve the app name** under Apps and games → New product → MSIX/PWA app.
3. From the reservation, note the **Identity** values (Product management → Product identity):
   - `Package/Identity/Name`
   - `Package/Identity/Publisher`
   - `Package/Properties/PublisherDisplayName`

## Set the identity

Set these in `src/mdviewx/Package.appxmanifest` so the produced MSIX matches the reservation. The
`<Identity />` and `<Properties />` elements are currently generated by the Uno SDK from the
project's `ApplicationId` / `ApplicationPublisher`; replace them with the exact Partner Center
values, e.g.:

```xml
<Identity Name="12345Publisher.mdviewx"
          Publisher="CN=ABCD1234-..."
          Version="1.0.0.0" />
<Properties>
  <DisplayName>mdviewx</DisplayName>
  <PublisherDisplayName>Your publisher display name</PublisherDisplayName>
  <Logo>Assets\StoreLogo.png</Logo>
</Properties>
```

> The MSIX `Version` must be `Major.Minor.Build.Revision` (e.g. `1.0.5.0`); the workflow passes
> `ApplicationDisplayVersion` which Uno maps into the package version.

## Submit

**Manual:** download the `.msix` from the GitHub Release and upload it in a new submission in
Partner Center.

**Automated (optional):** use the
[`microsoft/store-submission`](https://github.com/microsoft/store-submission) action (or the
`msstore` CLI) with an Azure AD app that has Partner Center API access. Add these repo secrets and
a publish step to `release.yml`:

- `MS_STORE_SELLER_ID`
- `MS_STORE_CLIENT_ID`
- `MS_STORE_TENANT_ID`
- `MS_STORE_CLIENT_SECRET`

## File association

`.md` / `.markdown` associations are already declared in `Package.appxmanifest`, so the installed
Store app is offered as a handler for Markdown files.
